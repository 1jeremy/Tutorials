Windows

  1. Download windows installer
  2. Double-click on the downloaded file

Skylight should install and launch automatically. To open Skylight when it’s not running, you may double-click the Skylight icon on your desktop.

MacOS

  1. Download mac installer
  2. Double click on the downloaded file
  3. Drag and drop Skylight over to your Applications folder

To open Skylight, you may double click Skylight icon in your Applications folder, or click the Skylight icon in your Launchpad.

Because Skylight launches at user login by default, a prompt of “Skylight wants access to control System Events” is expected on macOS 10.14 or newer. You can change this setting later in the Skylight preference settings.

Linux

  1. Download Skylight AppImage
  2. Right click on the downloaded AppImage file
  3. Select “Properties” in the pop-up menu
  4. In “Permissions” tab, check “Allow executing file as program”
  5. Double click on the AppImage file and then click “OK”

The process of making the AppImage file executable is required only for the first launch. After that, you could open Skylight by simply double click on Skylight.AppImage.
