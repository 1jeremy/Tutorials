BootCamp is divided into Courses, Lessons and Tasks. A Course groups together a super-category and is currently organized by asset-class. A Lesson focuses on the implementation of a single algorithm. A Task breaks up the algorithm implementing into tiny steps easily implemented, aiming to guide the user through the coding process.

  • Course: Algorithmic Trading with Equities
    • Lesson: Buy and Hold Algorithm
      • Task: Setting Cash

We have built a system for the community to design and submitting BootCamp Lessons and Tasks. If you think your content requires another Course please let us know and we'll create one for you.
