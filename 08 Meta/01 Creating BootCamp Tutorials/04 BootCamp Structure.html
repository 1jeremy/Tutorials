BootCamp is divided into Courses, Lessons, and Tasks. A Course groups together a super-category and is currently organized by asset-class. A Lesson focuses on the implementation of a single algorithm. A Task breaks up the algorithm implementation into tiny steps which can each be easily coded, with the aim of guiding the user through each layer of the algorithm design.

Core Principle

The central guiding principle of BootCamp lessons is they are focused on the implementation of a single algorithmic trading strategy. At each level of the system, the titles should be directly related to the implementation of the strategy, not the concepts it is covering.

  • Course: Algorithmic Trading with Equities
    • Lesson: Buy and Hold Algorithm
      • Task: Setting Cash

A lesson should be comprised of about 6-12 tasks, where each task builds upon the lessons learned from the previous task. We have built a system for the community to design and submit BootCamp Lessons and Tasks. If you think your content requires another Course please let us know and we'll create one for you.
